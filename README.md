# LoChord

A real-time MIDI chord controller for ESP32-S3 with chord generation, looping, and parameter control.


## Development Commands

Generate `compile_commands.json`:
```
pio run -t compiledb
```

Build and upload:
```
pio run -t upload
```

Serial monitor:
```
pio device monitor
```
Exit with `Ctrl+A` then `Ctrl+X`
